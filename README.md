# MUKER-online 

## Project Overview
MUKER-online is a web-based platform developed to digitize and organize high-end forest product catalogs, including Plywood, Edge-glued panels (Listonado), and fine timbers (Mahogany, Cedar, Oak). The project focuses on the logical structuring of technical specifications and materials to facilitate professional procurement and consultation.

Live Demo: [https://alejandrocis12.github.io/MUKER-online/](https://alejandrocis12.github.io/MUKER-online/)

## Mathematical Perspective & Information Logic
From my background in Applied Mathematics, I approached this development as a problem of classification and data optimization:

* **Taxonomic Data Structuring:** Implementation of a logical hierarchy to categorize a product database based on physical properties (density, wood type, and dimensions), optimizing user navigation through structured sets.
* **Component Optimization:** Designed an interface capable of managing high-resolution visual assets (wood textures) without compromising load speed, utilizing efficient resource management.
* **Interactivity Logic:** Development of a dynamic JavaScript-based carousel for brand and species exposure, prioritizing computational efficiency and minimal DOM manipulation.

## Tech Stack
* **Frontend:** Semantic HTML5 and CSS3 (Advanced use of Flexbox and Grid).
* **Interactivity:** Vanilla JavaScript (ES6+).
* **Design:** Responsive Web Design with a Mobile-First approach, tailored to the requirements of the industrial and technical carpentry sectors.

## Key Features
* **Categorized Inventory:** Rigorous classification between Plywood, Boards, and Precious Woods.
* **Dynamic Carousel:** Custom implementation for the visual navigation of species and suppliers.
* **Technical Specification Layout:** Structured distribution of dimensions and material origins.

## Academic Relevance
This project serves as practical evidence of applying logical structures to real-world commercial data. It demonstrates the ability to transition from theoretical organization to efficient computational implementation—a fundamental competency for graduate studies in Computer Science.
